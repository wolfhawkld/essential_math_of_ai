# 学习资源推荐

## 📚 书籍

### 线性代数
- **《线性代数及其应用》** - Gilbert Strang
  - MIT公开课配套教材，注重几何直觉
- **《深度学习中的线性代数》** - 在线资源
  - 专注于DL应用场景

### 微积分
- **《微积分之屠龙宝刀》** - 台湾交通大学公开课
  - 生动有趣的微积分入门
- **《Deep Learning Book - Chapter 4》** - Goodfellow
  - 专门针对DL的数值计算

### 概率统计
- **《概率论与数理统计》** - 陈希孺
  - 经典教材，理论严谨
- **《贝叶斯思维：统计建模的Python学习法》** - Allen B. Downey
  - 编程实践导向

### 优化理论
- **《凸优化》** - Stephen Boyd
  - 凸优化领域圣经，有配套公开课
- **《最优化导论》** - Edwin K. P. Chong
  - 工程应用导向

### 信息论
- **《信息论基础》** - Thomas M. Cover
  - 信息论经典教材
- **《深度学习》第3章** - Ian Goodfellow
  - DL视角的信息论

### 强化学习
- **《Reinforcement Learning: An Introduction》** - Sutton & Barto
  - RL圣经，2018第二版
- **《深度强化学习》** - 李宏毅
  - 台大公开课，深入浅出

## 💡 学习建议

### 数学基础薄弱？
1. 从 3Blue1Brown 的视频开始建立直觉
2. 用本项目的代码实践加深理解
3. 遇到不懂的公式查阅速查表

### 想深入理论？
1. 阅读推荐教材的相关章节
2. 完成教材习题
3. 实现论文中的算法

### 以应用为主？
1. 直接从 07_applications 开始
2. 遇到不理解的概念回查对应模块
3. 在Kaggle/Gymnasium上实践
